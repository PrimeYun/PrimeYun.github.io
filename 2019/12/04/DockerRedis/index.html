2、指定配置文件

Docker Hub 官网提供了两种方式使用自己的redis.conf：
1、构建自己的Dockerfile
2、使用docker run 的时候指定配置文件

这里使用第二种方式，指定配置文件。

（1）、获取配置文件
因为不同版本的redis配置文件可能有所不同，所以配置文件的版本需要与docker pull 拉取的镜像版本一致。
可以在redis官网或者使用wget命令下载redis，获取配置文件。
